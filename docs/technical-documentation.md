# Technical Documentation — Assignment 1

## Overview
This is a responsive personal portfolio website built with HTML, CSS, and JavaScript. It includes About, Projects, and Contact sections with basic interactivity.

## Tech Stack
- HTML5 (semantic structure)
- CSS3 (Grid/Flexbox + media queries)
- JavaScript (DOM manipulation, localStorage)

## Features
1. Responsive layout (desktop/tablet/mobile)
2. Theme toggle (dark/light) saved in localStorage
3. Greeting message based on time of day
4. Mobile navigation menu toggle
5. Contact form validation and success message (front-end only)

## How to Run Locally
Option 1:
- Open `index.html` directly in a browser

Option 2 (recommended):
- VS Code → install “Live Server” → right click `index.html` → “Open with Live Server”

## File Structure
- `index.html`: page layout
- `css/styles.css`: styles + responsive behavior
- `js/script.js`: interactivity
- `assets/images/`: images (placeholders)
- `docs/`: documentation

## Testing Checklist
- Resize browser to confirm responsive layout
- Toggle theme and refresh (check it persists)
- Test contact form validation (empty/invalid inputs)
- Test mobile menu open/close and link navigation
